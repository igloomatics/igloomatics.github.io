递归

数据结构与算法 — Sep 18, 2024

递归就是自己调用自己，需要有停止条件。

一层一层往下调用，再一层一层返回，

下面是几个经典的需要递归的算法：

Hanoi塔

题目背景: 一共3根柱子A、B、C，将所有盘子从第一根柱A子移到最后一根柱子V

def hanoi(n, A, B, C):
  % 终止条件
  if n==0:
     return

  hanoi(n-1, A, C, B) #将n-1个盘子由A通过C移动到B
  C.append(A.pop()) #将A上剩余一个盘子移动到C
  hanoi(n-1, B, A, C) #将B上的n-1个盘子通过A经由A移动到C

分治排序

def merge_sort(arr):
    # 如果数组只有一个元素或为空，直接返回
    if len(arr) <= 1:
        return arr

    # 找到数组的中间位置
    mid = len(arr) // 2

    # 递归地对左右两部分进行排序
    left_half = merge_sort(arr[:mid])
    right_half = merge_sort(arr[mid:])

    # 合并两个有序数组
    return merge(left_half, right_half)


def merge(left, right):
    sorted_arr = []
    i = j = 0

    # 按顺序比较两个数组中的元素，依次加入结果数组
    while i < len(left) and j < len(right):
        if left[i] < right[j]:
            sorted_arr.append(left[i])
            i += 1
        else:
            sorted_arr.append(right[j])
            j += 1

    # 将剩余元素（如果有）加入结果数组
    sorted_arr.extend(left[i:])
    sorted_arr.extend(right[j:])

    return sorted_arr
