KMP

保证主串的指针单调向前，保证了线性时间复杂度O(m+n)O(m+n)O(m+n)

What: PMT的概念

PMT数组中的值是字符串的真前缀集合与真后缀集合的交集中最长元素的长度，也是指针j回溯可到达的最近位置！

pmt[i]=max⁡length(\max length(maxlength(真前缀∩\cap∩真后缀)))

待匹配的子串有8个字符，那么PMT数组就会有8个值。

前缀："Hello"的前缀：{“H”,“He”,“Hel”,“Hell”}

后缀："World"的后缀：{“orld”,“rld”,“ld”,“d”}

e.g. “ababa”

前缀{“abab”,“aba”,“ab”,“a”}

后缀{“baba”,“aba”,“ba”,“a”}

前缀∩\cap∩后缀={“aba”,“a”}

pmt[5]=maxlength(前缀∩\cap∩后缀)=3

Why: 为什么引入PMT？

PMT可以用来确定指针j的位置，防止每次都回溯到开头i=0（i是上排文本串的指针)，造成O(mn)O(mn)O(mn)的逆天时间复杂度。

pmt[i]就是，从p[0]往后数、同时从p[i]往前数相同的位数，在保证前后缀相同的情况下，最多能数多少位。

这种情况下，失配，暴力BF的双层循环中，需要i=1 ，j=0，重新匹配，十分没有效率！

KMP中，我们希望上层指针i永远单调增加，不经历回溯。在i不动的情况下，希望j回退至一个合适的位置（等价于移动模式串p），使得ababa的真后缀与ababc的真前缀相匹配，且j能指示匹配位数。

如图，“ABACABA"的后缀"ABA” 与 "ABAC"的前缀"ABA"相匹配，无需再次验证，只需验证后面的B?=C

如图，i不动，j=pmt[j-1]=pmt[7-1]=pmt[6]=3。i=7，j=3，B!=C，失配。

如图，i不动，继续回溯j，j=pmt[j-1]=pmt[3-1]=pmt[2]=1。i=7，j=1，B==B，匹配！

i++，j++，发现一直可以匹配到末尾。

How: 用代码求出PMT

在错开一位后，让模式串p自己匹配自己（这相当于是用前缀去匹配后缀）。我们知道pmt[0]=0，而之后的每一位则可以通过在匹配过程中记录j值得到。

void get_pmt(const string& s) {
    for (int i = 1, j = 0; i < s.length(); ++i) {
        while (j != 0 && s[i] != s[j]) j = pmt[j - 1]; //字符失配，指针j回溯
        if (s[i] == s[j]) j++; //字符匹配，指针j向前
        pmt[i] = j;
    }
}

匹配

int kmp(const string& s, const string& p) {
    for (int i = 0, j = 0; i < s.length(); ++i) {
      //i是目标串指针 j是模式串指针
        while (j && s[i] != p[j]) j = pmt[j - 1]; //字符失配，指针j回溯
        if (s[i] == p[j]) j++; //字符匹配，指针j向前
        if (j == p.length()) {
            return i - j + 2  // 首个完全匹配上的位置，i-(j-1)+1,指针j处于已经匹配完的模式串的最后一个字符的下一个
        }
    }
}

#include<iostream>
using namespace std;
const int MAXN=1e6+5;
int pmt[MAXN];
void get_pmt(const string& s) {
    for (int i = 1, j = 0; i < s.length(); ++i) {
        while (j != 0 && s[i] != s[j]) j = pmt[j - 1]; //字符失配，指针j回溯
        if (s[i] == s[j]) j++; //字符匹配，指针j向前
        pmt[i] = j;
    }
}
int kmp(const string& s, const string& p) {
    for (int i = 0, j = 0; i < s.length(); ++i) {
      //i是目标串指针 j是模式串指针
        while (j && s[i] != p[j]) j = pmt[j - 1]; //字符失配，指针j回溯
        if (s[i] == p[j]) j++; //字符匹配，指针j向前
        if (j == p.length()) {
            return i - j + 2; // 首个完全匹配上的位置，i-(j-1)+1,指针j处于已经匹配完的模式串的最后一个字符的下一个
        }
    }
}
int main()
{
    string s, p;
    cin >> s >> p;
    get_pmt(p);
    cout<<kmp(s,p)<<endl;
    return 0;
}
